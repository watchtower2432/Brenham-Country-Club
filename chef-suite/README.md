## üöÄ Quick Start

### For End Users (Installation)

1. **Download** the installer from your IT department or the provided link
2. **Double-click** `ChefSuite-Setup-1.0.0.exe` to begin installation
3. **Follow** the installation wizard (recommended settings are pre-selected)
4. **Launch** Chef Suite from your desktop shortcut or Start menu
5. **Login** with your credentials (contact your administrator for login details)

### For Administrators (Deployment)

```bash
# Clone the repository
git clone https://github.com/watchtower2432/Brenham-Country-Club.git
cd Brenham-Country-Club/chef-suite

# Build the desktop application
chmod +x build-desktop.sh
./build-desktop.sh

# The installer will be created in frontend/dist/
```
